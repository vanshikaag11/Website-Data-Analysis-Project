# 📊 Website Data Analysis Project

## 📌 Project Overview

This project is an interactive **Website Performance Analysis Dashboard** developed using **Python**, **Jupyter Notebook**, and **Streamlit**.

The project analyzes website traffic and engagement data to generate meaningful insights through data visualization and dashboard techniques.

The dashboard helps in understanding:

- User traffic patterns
- Engagement performance
- Traffic sources
- Sessions analysis
- Website engagement trends

---

# 🚀 Project Workflow

## Step 1 — Data Analysis in Jupyter Notebook

Initially, the complete data analysis process was performed in a **Jupyter Notebook (.ipynb)** environment.

The notebook included:

- Data loading
- Data cleaning
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Insight generation

Libraries used during analysis:

- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Step 2 — Conversion to Python Script

After completing the analysis in Jupyter Notebook, the notebook was exported into a Python script (`.py`) file using VS Code/Jupyter export functionality.

The notebook code was then restructured and optimized for web application development.

---

## Step 3 — Streamlit Web Application Development

The analyzed data and visualizations were converted into an interactive web dashboard using **Streamlit**.

The Streamlit application provides:

- Interactive dashboard interface
- Dynamic visualizations
- Sidebar filters
- Performance metrics
- Engagement tracking
